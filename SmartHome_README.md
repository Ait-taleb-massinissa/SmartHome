# SmartHome ESP32 Project (ESP-IDF)

This project is an ESP32-based embedded system built using the ESP-IDF framework.  
It includes multiple modules such as sensors, communication, and system synchronization.

---

# Requirements

Before running this project, install the following:

- Visual Studio Code
- ESP-IDF Extension for VSCode
- Git
- Python 3.8+
- USB driver (CP210x or CH340 if needed) [download](https://www.silabs.com/documents/public/software/CP210x_VCP_Windows.zip)

---

# 1. ESP-IDF Setup (VSCode)

## Install ESP-IDF Extension
Open VSCode → Extensions → install:

Espressif IDF

---

## Configure ESP-IDF

Press:
Ctrl + Shift + P

Then run:
ESP-IDF: Configure ESP-IDF extension

Choose:
- Express setup (recommended)
- ESP-IDF v5.x
- Install all tools (Python, CMake, Ninja, Git)
